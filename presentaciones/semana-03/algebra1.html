¿Qué es SymPy?

Matemática simbólica en Python

Librería de álgebra computacional

SymPy permite hacer operaciones matemáticas SIMBÓLICAS (no numéricas). Manipula expresiones algebraicas como lo harías en papel, pero sin errores.

Nota: En este curso usaremos "from sympy import *" para simplificar, pero en producción es mejor importar solo lo necesario.

¿Por qué init_printing()?

Esta función hace que las expresiones se muestren con formato matemático bonito (como en LaTeX) en lugar de texto plano. Transforma "x**2 + 3*x" en "x² + 3x".

Crear símbolos: el primer paso

Variables matemáticas en Python

# Método 1: Un símbolo x = symbols('x') print(type(x)) # <class 'sympy.core.symbol.Symbol'> # Método 2: Varios símbolos x, y, z = symbols('x y z') # Método 3: Con restricciones n = Symbol('n', integer=True, positive=True) theta = Symbol('theta', real=True)
# Método 4: Secuencias x1, x2, x3 = symbols('x1:4') # Crea x1, x2, x3 a, b, c = symbols('a:d') # Crea a, b, c # Crear 10 coeficientes coefs = symbols('c0:10') print(coefs) # (c0, c1, c2, ..., c9)

Importante: diferencia entre símbolo y variable de Python

En "x = symbols('x')", la variable de Python 'x' contiene un símbolo matemático 'x'. El nombre no tiene que coincidir: podrías hacer "variable = symbols('x')", pero sería confuso.

# Ejemplo de uso con símbolos creados x, y = symbols('x y') # Ahora x e y son símbolos matemáticos expresion = x**2 + 3*x + y print(expresion) # Salida: x² + 3x + y (con formato bonito si usaste init_printing)

Expandir expresiones

Productos notables automáticamente

from sympy import * x, y = symbols('x y') # =================================== # 1. EXPANDIR: (a+b)² → a² + 2ab + b² # =================================== expr1 = (x + 3)**2 print("Original:", expr1) # (x + 3)² print("Expandido:", expand(expr1)) # x² + 6x + 9 # =================================== # 2. CUADRADO DE DIFERENCIA # =================================== expr2 = (x - 5)**2 print(expand(expr2)) # x² - 10x + 25 # =================================== # 3. DIFERENCIA DE CUADRADOS # =================================== expr3 = (x + y) * (x - y) print(expand(expr3)) # x² - y² # =================================== # 4. CUBO DE SUMA # =================================== expr4 = (x + 2)**3 print(expand(expr4)) # x³ + 6x² + 12x + 8

expand() aplica todas las reglas algebraicas para escribir la expresión en su forma más desarrollada.

Factorizar expresiones

El proceso inverso de expandir

from sympy import * x = symbols('x') # =================================== # 1. DIFERENCIA DE CUADRADOS # =================================== expr1 = x**2 - 4 print(factor(expr1)) # (x - 2)(x + 2) # =================================== # 2. TRINOMIO CUADRADO PERFECTO # =================================== expr2 = x**2 + 6*x + 9 print(factor(expr2)) # (x + 3)² # =================================== # 3. FACTORIZACIÓN GENERAL # =================================== expr3 = x**3 - x**2 + x - 1 print(factor(expr3)) # (x - 1)(x² + 1) # =================================== # 4. FACTORIZAR COMPLETAMENTE # =================================== expr4 = x**4 - 16 print(factor(expr4)) # (x - 2)(x + 2)(x² + 4)

expand() vs factor()

Son operaciones inversas. expand() desarrolla productos, factor() encuentra factores comunes.

# Verificar que son inversos original = (x + 3)**2 expandida = expand(original) de_vuelta = factor(expandida) print(original == de_vuelta) # True

Simplificar expresiones

Reducir a la forma más simple

from sympy import * x = symbols('x') # =================================== # 1. SIMPLIFICAR IDENTIDADES TRIGONOMÉTRICAS # =================================== expr1 = sin(x)**2 + cos(x)**2 print(simplify(expr1)) # 1 # =================================== # 2. SIMPLIFICAR FRACCIONES # =================================== expr2 = (x**2 - 1) / (x - 1) print(simplify(expr2)) # x + 1 # =================================== # 3. SIMPLIFICAR EXPONENTES # =================================== expr3 = x**3 * x**4 / x**2 print(simplify(expr3)) # x⁵ # =================================== # 4. SIMPLIFICACIÓN ESPECÍFICA # =================================== # trigsimp: solo trigonometría print(trigsimp(sin(x)**2 + cos(x)**2)) # 1 # radsimp: racionalizar denominadores expr4 = 1 / (sqrt(2) + sqrt(3)) print(radsimp(expr4)) # -√2 + √3

simplify() intenta varias estrategias y devuelve la expresión más simple. Es "inteligente" pero puede ser lento con expresiones grandes.

Resolver ecuaciones

solve() y solveset()

# MÉTODO TRADICIONAL: solve() from sympy import * x = symbols('x') # Ecuación: x² - 4 = 0 sol = solve(x**2 - 4, x) print(sol) # [-2, 2] # Sistema de ecuaciones y = symbols('y') sistema = solve([ x + y - 5, x - y - 1 ], [x, y]) print(sistema) # {x: 3, y: 2}
# MÉTODO MODERNO: solveset() from sympy import * x = symbols('x') # Retorna conjunto sol = solveset( x**2 - 4, x, domain=S.Reals ) print(sol) # FiniteSet(-2, 2) # Maneja infinitas mejor sol2 = solveset( sin(x), x, domain=S.Reals ) # ImageSet con ∞ soluciones

Diferencias clave

solve(): Retorna lista, puede resolver sistemas (múltiples variables). Usa para sistemas de ecuaciones.

solveset(): Retorna conjunto matemático, una variable solamente, maneja mejor casos con infinitas soluciones. Más moderno y recomendado para ecuaciones simples.

Derivación simbólica

Calcular derivadas exactas

from sympy import * x = symbols('x') # =================================== # 1. DERIVADA SIMPLE # =================================== f = x**3 + 2*x**2 + 5 df = diff(f, x) print(f"f(x) = {f}") print(f"f'(x) = {df}") # f'(x) = 3x² + 4x # =================================== # 2. SEGUNDA DERIVADA # =================================== ddf = diff(f, x, 2) # Derivar 2 veces print(f"f''(x) = {ddf}") # f''(x) = 6x + 4 # =================================== # 3. DERIVADAS PARCIALES # =================================== x, y = symbols('x y') g = x**2 * y + 3*x*y**2 dgdx = diff(g, x) # ∂g/∂x dgdy = diff(g, y) # ∂g/∂y print(f"∂g/∂x = {dgdx}") # 2xy + 3y² print(f"∂g/∂y = {dgdy}") # x² + 6xy

diff() calcula derivadas EXACTAS. No hay errores de aproximación numérica.

Aplicación: derivar función de pérdida MSE

Gradientes para gradient descent

from sympy import * # Definir símbolos x, y, m, b = symbols('x y m b') # Modelo de predicción lineal: ŷ = mx + b y_pred = m * x + b # Error cuadrático para UN punto: (y - ŷ)² error = (y - y_pred)**2 print("Error cuadrático:") print(expand(error)) # Salida: b² + 2bm·x + m²x² - 2by - 2mxy + y² # =================================== # DERIVADAS PARCIALES (GRADIENTES) # =================================== # ∂Error/∂m (gradiente respecto a pendiente) d_error_dm = diff(error, m) print(" ∂Error/∂m =") print(simplify(d_error_dm)) # 2x(b + mx - y) # ∂Error/∂b (gradiente respecto a intercepto) d_error_db = diff(error, b) print(" ∂Error/∂b =") print(simplify(d_error_db)) # 2(b + mx - y) # =================================== # CONVERTIR A FUNCIÓN NUMÉRICA # =================================== import numpy as np # lambdify convierte expresión simbólica → función Python grad_m_func = lambdify([x, y, m, b], d_error_dm, "numpy") grad_b_func = lambdify([x, y, m, b], d_error_db, "numpy") # Usar con datos reales X_data = np.array([1, 2, 3, 4, 5]) y_data = np.array([2, 4, 6, 8, 10]) m_val, b_val = 1.5, 0.5 gradientes_m = grad_m_func(X_data, y_data, m_val, b_val) gradientes_b = grad_b_func(X_data, y_data, m_val, b_val) print(f" Gradiente promedio ∂E/∂m: {np.mean(gradientes_m):.3f}") print(f"Gradiente promedio ∂E/∂b: {np.mean(gradientes_b):.3f}")

lambdify: de símbolos a funciones ejecutables

Puente entre SymPy y NumPy

¿Qué hace lambdify?

Convierte expresiones simbólicas de SymPy en funciones de Python que pueden evaluarse con NumPy (rápido, vectorizado).

from sympy import * import numpy as np import matplotlib.pyplot as plt # =================================== # 1. CREAR EXPRESIÓN SIMBÓLICA # =================================== x = symbols('x') expr = sin(x) * cos(x) # =================================== # 2. CONVERTIR A FUNCIÓN NUMPY # =================================== f_numpy = lambdify(x, expr, "numpy") # Ahora f_numpy es una función Python normal print(type(f_numpy)) # <class 'function'> # =================================== # 3. EVALUAR EN ARRAYS (VECTORIZADO) # =================================== x_vals = np.linspace(0, 2*np.pi, 1000) y_vals = f_numpy(x_vals) # ¡Muy rápido! # =================================== # 4. GRAFICAR # =================================== plt.figure(figsize=(10, 5)) plt.plot(x_vals, y_vals, color='#FF8C00', linewidth=2) plt.title('f(x) = sin(x)·cos(x)', fontsize=14) plt.xlabel('x') plt.ylabel('f(x)') plt.grid(True, alpha=0.3) plt.show()

lambdify es esencial cuando necesitas evaluar expresiones simbólicas millones de veces (como en ML).

Ventajas de SymPy para este curso

Instalación trivial en Google Colab, integración perfecta con NumPy, Pandas y Matplotlib, curva de aprendizaje suave.

Limitaciones de SymPy

No tan rápido como Mathematica, limitado para álgebra computacional muy avanzada, no tiene interfaz gráfica nativa.

Para ciencia de datos y machine learning, SymPy es la opción ideal por su integración nativa con el ecosistema Python.
