# Presentaciones del curso

Esta carpeta contiene las presentaciones interactivas en HTML para cada semana del curso.

## Estructura

Cada semana tiene su propia carpeta con el formato `semana-XX/` donde se almacenan:
- Presentación principal en HTML
- Recursos visuales adicionales (si aplica)
- Notas del instructor (si aplica)

## Acceso

Las presentaciones pueden visualizarse directamente desde el navegador abriendo los archivos `.html` o a través del sitio web del curso en GitHub Pages.
