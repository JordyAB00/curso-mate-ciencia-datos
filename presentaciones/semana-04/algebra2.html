Productos notables con SymPy

expand() hace el trabajo pesado

Función expand()

Toma una expresión factorizada y la desarrolla completamente aplicando todas las reglas algebraicas. Es el equivalente a multiplicar término por término, pero automático y sin errores.

from sympy import * x, y = symbols('x y') # =========================================== # PRODUCTOS NOTABLES BÁSICOS # =========================================== # 1. Cuadrado de suma: (a + b)² = a² + 2ab + b² expr1 = (x + 3)**2 print(expand(expr1)) # Salida: x**2 + 6*x + 9 # 2. Cuadrado de diferencia: (a - b)² = a² - 2ab + b² expr2 = (2*x - 5)**2 print(expand(expr2)) # Salida: 4*x**2 - 20*x + 25 # 3. Diferencia de cuadrados: (a+b)(a-b) = a² - b² expr3 = (x + y) * (x - y) print(expand(expr3)) # Salida: x**2 - y**2 # 4. Cubo de suma: (a+b)³ expr4 = (x + 2)**3 print(expand(expr4)) # Salida: x**3 + 6*x**2 + 12*x + 8 # 5. Expresiones multivariables expr5 = (x + y + 1)**2 print(expand(expr5)) # Salida: x**2 + 2*x*y + 2*x + y**2 + 2*y + 1

expand() es perfecto para verificar manualmente tu trabajo algebraico. Si expandiste a mano, usa SymPy para confirmar.

Factorización automática

factor() encuentra factores irreducibles

from sympy import * x = symbols('x') # ============================= # DIFERENCIA DE CUADRADOS # ============================= expr1 = x**2 - 9 print(factor(expr1)) # (x - 3)*(x + 3) # ============================= # TRINOMIO CUADRADO PERFECTO # ============================= expr2 = x**2 + 6*x + 9 print(factor(expr2)) # (x + 3)**2 # ============================= # FACTORIZACIÓN GENERAL # ============================= expr3 = x**2 + 5*x + 6 print(factor(expr3)) # (x + 2)*(x + 3)
# ============================= # DIFERENCIA/SUMA DE CUBOS # ============================= expr4 = x**3 - 8 print(factor(expr4)) # (x - 2)*(x**2 + 2*x + 4) expr5 = x**3 + 27 print(factor(expr5)) # (x + 3)*(x**2 - 3*x + 9) # ============================= # FACTORIZACIÓN COMPLETA # ============================= expr6 = x**4 - 16 print(factor(expr6)) # (x - 2)*(x + 2)*(x**2 + 4)

expand() vs factor(): operaciones inversas

Son funciones complementarias. expand() desarrolla, factor() descompone. Úsalas en conjunto para verificar identidades algebraicas.

# Verificar que son inversas original = (x + 5) * (x - 3) expandida = expand(original) # x**2 + 2*x - 15 factorizada = factor(expandida) # (x - 3)*(x + 5) print(original == factorizada) # True

Teorema del binomio en Python

Coeficientes binomiales y distribución binomial

Recordatorio matemático

(a + b)ⁿ = Σ C(n,k) · aⁿ⁻ᵏ · bᵏ donde C(n,k) = n! / [k!(n-k)!]

# ================================ # COEFICIENTES BINOMIALES # ================================ from sympy import binomial, expand from sympy import symbols x, y = symbols('x y') # Calcular C(5,2) print(binomial(5, 2)) # Salida: 10 # Calcular C(10,3) print(binomial(10, 3)) # Salida: 120 # Triángulo de Pascal (fila 4) for k in range(5): print(binomial(4, k), end=" ") # Salida: 1 4 6 4 1
# ================================ # EXPANSIÓN BINOMIAL # ================================ # Expandir (x + y)⁴ expr = (x + y)**4 print(expand(expr)) # x**4 + 4*x**3*y + 6*x**2*y**2 + # 4*x*y**3 + y**4 # Verificar coeficientes for k in range(5): coef = binomial(4, k) print(f"Coef x^{4-k}y^{k}: {coef}") # Coef x^4y^0: 1 # Coef x^3y^1: 4 # Coef x^2y^2: 6 # ...

Conexión con ciencia de datos: distribución binomial

La distribución binomial P(X=k) = C(n,k)·p^k·(1-p)^(n-k) modela eventos binarios (éxito/fracaso). Se usa en A/B testing, clasificación binaria, simulaciones Monte Carlo.

from scipy.stats import binom import numpy as np # Probabilidad de obtener exactamente 5 caras en 10 lanzamientos prob = binom.pmf(k=5, n=10, p=0.5) print(f"P(X=5) = {prob:.4f}") # 0.2461 # Generar 1000 muestras muestras = np.random.binomial(n=10, p=0.5, size=1000) print(f"Media: {muestras.mean():.2f}, Esperada: {10*0.5}")

División de polinomios con SymPy

div() y verificación de factores

División algebraica: A(x) = B(x)·Q(x) + R(x)

SymPy puede dividir polinomios y retornar cociente y resto simultáneamente.

from sympy import symbols, div, Poly x = symbols('x') # =================================================== # DIVISIÓN BÁSICA # =================================================== # Dividir: (x² - 4) / (x - 2) dividendo = x**2 - 4 divisor = x - 2 cociente, resto = div(dividendo, divisor, domain='ZZ') print(f"Cociente: {cociente}") # x + 2 print(f"Resto: {resto}") # 0 # =================================================== # VERIFICACIÓN: dividendo = divisor*cociente + resto # =================================================== verificacion = divisor * cociente + resto print(verificacion == dividendo) # True # =================================================== # DIVISIÓN CON RESTO NO CERO # =================================================== f = 5*x**2 + 10*x + 3 g = 2*x + 2 q, r = div(f, g, domain='QQ') # QQ = racionales print(f"q = {q}") # 5*x/2 + 5/2 print(f"r = {r}") # -2

Teorema del resto

El resto de dividir p(x) entre (x-c) es p(c). No necesitas dividir, solo evalúa.

# Resto de (x³+2x²-5x+2)/(x-1) p = x**3 + 2*x**2 - 5*x + 2 # Método 1: dividir q, r = div(p, x-1) print(f"Resto: {r}") # 0 # Método 2: evaluar p(1) print(p.subs(x, 1)) # 0

Si el resto es 0, entonces el divisor es un factor. Esto nos lleva al teorema del factor: (x-c) es factor de p(x) si y solo si p(c) = 0.

Encontrar ceros: roots() y solve()

Raíces simbólicas exactas

Teorema fundamental del álgebra en acción

Todo polinomio de grado n tiene exactamente n raíces complejas (contando multiplicidades). SymPy encuentra estas raíces simbólicamente cuando es posible.

from sympy import symbols, roots from sympy import solve, real_roots x = symbols('x') # ============================= # FUNCIÓN roots() # ============================= # Retorna diccionario {raíz: multiplicidad} p1 = x**2 - 4 print(roots(p1)) # {-2: 1, 2: 1} p2 = x**2 - 6*x + 9 print(roots(p2)) # {3: 2} ← raíz doble p3 = x**3 - 1 print(roots(p3)) # {1: 1, -1/2 - sqrt(3)*I/2: 1, # -1/2 + sqrt(3)*I/2: 1}
# ============================= # FUNCIÓN solve() # ============================= # Retorna lista de soluciones p1 = x**2 - 4 print(solve(p1, x)) # [-2, 2] p2 = x**3 - 6*x**2 + 11*x - 6 print(solve(p2, x)) # [1, 2, 3] # ============================= # SOLO RAÍCES REALES # ============================= p3 = x**3 - 1 print(real_roots(p3)) # [1] ← solo la real

Diferencia entre roots() y solve()

roots(): Retorna diccionario con multiplicidades. Más información sobre la estructura del polinomio.

solve(): Retorna lista simple de soluciones. Más fácil para usar en código.

# Ejemplo: encontrar y verificar raíces polinomio = x**3 - 7*x + 6 raices = solve(polinomio, x) print("Raíces:", raices) # [-3, 1, 2] # Verificar cada raíz for r in raices: resultado = polinomio.subs(x, r) print(f"p({r}) = {resultado}") # Todos dan 0

Raíces numéricas con NumPy

Cuando necesitas números, no símbolos

numpy.roots() vs SymPy roots()

NumPy encuentra raíces NUMÉRICAS (aproximaciones en punto flotante) usando eigenvalores de la matriz compañera. Es extremadamente rápido pero pierde exactitud simbólica.

# =========================== # NumPy: RÁPIDO, NUMÉRICO # =========================== import numpy as np # Coeficientes de x³ - 1 # Orden DESCENDENTE: [1, 0, 0, -1] coefs = [1, 0, 0, -1] raices = np.roots(coefs) print(raices) # [ 1. +0.j ] # [-0.5+0.8660254j ] # [-0.5-0.8660254j ] # Trabajar solo con reales raices_reales = raices[ np.isreal(raices) ].real print(raices_reales) # [1.]
# =========================== # SymPy: EXACTO, SIMBÓLICO # =========================== from sympy import * x = symbols('x') p = x**3 - 1 raices = roots(p) print(raices) # {1: 1, # -1/2 - sqrt(3)*I/2: 1, # -1/2 + sqrt(3)*I/2: 1} # Expresiones EXACTAS ↑ # No hay error de redondeo # Convertir a numérico si necesitas from sympy import N for r in raices.keys(): print(N(r, 5)) # 1.0000 # -0.50000 - 0.86603*I # -0.50000 + 0.86603*I
Característica NumPy roots() SymPy roots()
Tipo de resultado Aproximación numérica (float64) Expresión simbólica exacta
Velocidad Muy rápido (matriz compañera) Más lento (álgebra simbólica)
Precisión ~15 dígitos significativos Exacta (sin error)
Uso ideal Cálculos numéricos masivos Verificación, educación, análisis

Regla práctica: usa SymPy para ENTENDER, usa NumPy para CALCULAR en producción.

Aplicación: eigenvalores y PCA

Los ceros de polinomios en machine learning

Conexión matemática: eigenvalores como raíces

Los eigenvalores de una matriz A son las raíces del polinomio característico det(A - λI) = 0. Por el teorema fundamental del álgebra, toda matriz n×n tiene exactamente n eigenvalores complejos.

import numpy as np from sympy import * # ========================================================= # PARTE 1: CÁLCULO SIMBÓLICO DEL POLINOMIO CARACTERÍSTICO # ========================================================= lam = symbols('lambda') # Matriz 2×2 A_sym = Matrix([[4, -2], [1, 1]]) # Matriz identidad I = eye(2) # Polinomio característico: det(A - λI) poli_caract = (A_sym - lam*I).det() print("Polinomio característico:") print(poli_caract) # lambda**2 - 5*lambda + 6 # Factorizar print(factor(poli_caract)) # (lambda - 2)*(lambda - 3) # Eigenvalores = raíces eigenvals_sym = solve(poli_caract, lam) print("Eigenvalores simbólicos:", eigenvals_sym) # [2, 3] # ========================================================= # PARTE 2: VERIFICACIÓN NUMÉRICA CON NUMPY # ========================================================= A_num = np.array([[4, -2], [1, 1]]) eigenvals_num, eigenvecs = np.linalg.eig(A_num) print("Eigenvalores numéricos:", eigenvals_num) # [3. 2.] print(" Eigenvectores:") print(eigenvecs) # [[ 0.89442719 -0.70710678] # [ 0.4472136 0.70710678]]

¿Por qué esto importa en ML?

PCA (Principal Component Analysis): Encuentra direcciones de máxima varianza usando eigenvectores de la matriz de covarianza. Los eigenvalores más grandes indican las componentes principales más importantes.

PCA: reducción dimensional con eigenvalores

Álgebra polinomial aplicada a datos reales

from sklearn.decomposition import PCA from sklearn.datasets import load_iris import numpy as np import matplotlib.pyplot as plt # ========================================================= # CARGAR DATOS: IRIS DATASET (4 DIMENSIONES) # ========================================================= iris = load_iris() X = iris.data # 150 muestras × 4 características y = iris.target print(f"Dimensiones originales: {X.shape}") # (150, 4) # ========================================================= # APLICAR PCA: REDUCIR A 2 COMPONENTES PRINCIPALES # ========================================================= pca = PCA(n_components=2) X_reducido = pca.fit_transform(X) print(f"Dimensiones reducidas: {X_reducido.shape}") # (150, 2) # ========================================================= # ANALIZAR EIGENVALORES (VARIANZA EXPLICADA) # ========================================================= print(" Eigenvalores (varianza explicada):") print(pca.explained_variance_) # [4.22824171 0.24267075] print(" Porcentaje de varianza explicada:") print(pca.explained_variance_ratio_) # [0.92461872 0.05306648] # ↑ Las 2 componentes capturan 97.77% de la información print(f" Varianza total capturada: {pca.explained_variance_ratio_.sum():.2%}") # 97.77% # ========================================================= # VISUALIZAR EN 2D # ========================================================= plt.figure(figsize=(10, 6)) colores = ['#FF8C00', '#2d2d2d', '#6e6e73'] for i in range(3): plt.scatter( X_reducido[y == i, 0], X_reducido[y == i, 1], c=colores[i], label=iris.target_names[i], s=50, alpha=0.7 ) plt.xlabel('Primera componente principal (PC1)', fontsize=12) plt.ylabel('Segunda componente principal (PC2)', fontsize=12) plt.title('PCA de Iris Dataset: 4D → 2D', fontsize=14, fontweight='bold') plt.legend() plt.grid(True, alpha=0.3) plt.show()

Conexión clave: PCA calcula eigenvalores del polinomio característico de la matriz de covarianza. Los eigenvalores más grandes = direcciones de mayor varianza = componentes principales más importantes.

Cuándo usar cada función de SymPy

Guía de decisión rápida

Función Qué hace Cuándo usarla Ejemplo
expand() Desarrolla productos, aplica distributiva Cuando tienes factores y quieres la forma desarrollada (x+3)² → x²+6x+9
factor() Descompone en factores irreducibles Para resolver ecuaciones, simplificar, encontrar ceros x²-9 → (x-3)(x+3)
simplify() Reduce a forma más simple (heurístico) Expresiones complicadas que quieres limpiar sin²(x)+cos²(x) → 1
div() División polinomial (cociente + resto) División larga algebraica, verificar factores (x²-4)/(x-2) → (x+2, 0)
roots() Raíces con multiplicidades (dict) Análisis estructural del polinomio x²-6x+9 → {3: 2}
solve() Soluciones de ecuaciones (lista) Resolver ecuaciones para usar en código x²-4=0 → [-2, 2]

Flujo de trabajo típico

1. Define el polinomio
2. Factoriza con factor()
3. Encuentra ceros con solve()
4. Verifica con subs() o div()
5. Expande con expand() si necesitas

# Ejemplo de flujo completo from sympy import * x = symbols('x') p = x**2 - 5*x + 6 print("1. Original:", p) print("2. Factorizado:", factor(p)) print("3. Ceros:", solve(p)) print("4. Verificar p(2):", p.subs(x,2))

Lo que dominamos hoy

Productos notables y factorización

expand() desarrolla expresiones automáticamente. factor() encuentra la descomposición en factores irreducibles. Son operaciones inversas fundamentales para simplificar álgebra.

Teorema del binomio

roots() y solve() encuentran raíces simbólicas. NumPy roots() para aproximaciones numéricas rápidas. Los eigenvalores son ceros del polinomio característico, fundamentales en PCA y ML.
